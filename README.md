# hubot-ebichan

main brain of ebichan

See [`src/ebichan.coffee`](src/ebichan.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-ebichan --save`

Then add **hubot-ebichan** to your `external-scripts.json`:

```json
[
  "hubot-ebichan"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
